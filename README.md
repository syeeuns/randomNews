# randomNews
Jquery, Ajax, Bulma css 이용해서 랜덤뉴스 페이지 제작 (neswAPI 이용)<br>

간단한 프로그램은 JS, CSS가 짧아서 한 페이지에 다 넣음

키워드 넣어주면 키워드 관련 랜덤뉴스 띄워줌 <br>
**ajax url에도 템플릿 리터럴로 주소 넣어줄 수 있음**
